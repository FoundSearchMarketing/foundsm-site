# FoundSM Component Catalog

Use the preferred POC blocks for new reusable page-builder work:

- `modernHeroBlock`: first viewport, campaign promise, primary CTA.
- `splitFeatureBlock`: one idea with supporting media.
- `featureTabsBlock`: parallel methods, capabilities, or comparisons.
- `cardGridBlock`: benefits, proof points, services, or steps.
- `statementBandBlock`: emphasized strategic statement.
- `modernCtaBlock`: next action.

Existing generic blocks remain valid: `heroBlock`, `statsBlock`, `featuresBlock`, `testimonialBlock`, `ctaBlock`, `formBlock`, `textBlock`, `imageTextBlock`, `logoBarBlock`, and `faqBlock`.

When adding editability, expose structured fields and tokens. Do not expose raw CSS controls unless the user explicitly asks and the risk is called out.
